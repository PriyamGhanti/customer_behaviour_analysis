# customer_behaviour_analysis
Customer Behavior Analysis project using Python, SQL, and data visualization. Performed data cleaning and EDA with Pandas and NumPy, executed SQL queries in MS SQL Server to uncover business insights, and built an interactive dashboard to visualize customer trends, purchasing patterns, retention metrics, and key KPIs.


Project Overview

This project focuses on analyzing customer behavior to uncover actionable business insights. Using Python, SQL, and data visualization techniques, the project explores customer purchasing patterns, retention trends, and key performance metrics to support data-driven decision-making.

Objectives
Understand customer purchasing behavior.
Identify trends and patterns in customer activity.
Generate business insights using SQL queries.
Visualize key metrics through an interactive dashboard.
